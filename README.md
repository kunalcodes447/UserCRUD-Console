UserCRUD-Console

UserCRUD-Console is a simple Java console application that performs basic CRUD (Create, Read, Update, Delete) operations on user data. The program interacts with the user through the terminal, allowing easy management of user records in memory.

The application uses core Java features such as collections, loops, and conditional logic to store and manipulate user information like name, email, and age. It provides a text-based menu where users can add new entries, view all records, update existing users by ID, delete records, and exit the program.

Features

Add (Create) new user records

Display (Read) all users

Update user details by ID

Remove (Delete) user records

Console-based menu with user prompts

How It Works

When the application starts, it displays a menu with options to perform CRUD operations. Based on user input, the program executes the corresponding method using an in-memory data structure (such as ArrayList). User records are stored as objects, and the program continually loops until the user decides to exit.

Technologies Used

Java SE

Collections (ArrayList)

OOP Concepts (Classes & Objects)

Console Input/Output
